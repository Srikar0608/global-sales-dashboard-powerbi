# 📊 Global Sales Performance Dashboard

## 🔹 Project Overview

This project presents an interactive Power BI dashboard analyzing global sales performance across different countries and products. The dashboard provides insights into sales, profit, discounts, and product-level performance.

## 🔹 Key Features

* 🌍 Sales distribution by country (Map visualization)
* 📦 Product-wise sales and units analysis
* 💰 Profit comparison across countries
* 📉 Discount impact on overall sales
* 📊 Interactive filters for dynamic analysis

## 🔹 Tools & Technologies

* Power BI
* Excel / CSV
* Data Cleaning & Transformation

## 🔹 Key Insights

* The United States and Canada contribute significantly to overall sales
* Paseo product has the highest sales share
* France shows the highest profit among countries
* Discounts have a noticeable impact on total revenue
